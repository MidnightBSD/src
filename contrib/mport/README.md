# mport
MidnightBSD Package Manager

## Requirements
Will run on MidnightBSD 1.2 and higher. 

Depends on:
* sqlite3
* libarchive
* bzip2
* lzma

Versions prior to 2.1.0 also depended on
* libdispatch
* Blocksruntime
