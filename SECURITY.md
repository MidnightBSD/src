# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 3.2     | :white_check_mark: |
| 3.1     | :white_check_mark: |
| 3.0     | :x: |
| < 2.2   | :x:                |

## Reporting a Vulnerability

Report security vulnerabilities to security@midnightbsd.org or create a pull request at https://github.com/MidnightBSD/midnightbsd-vulns
